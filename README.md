# 🚕 OLA Sales Dashboard

## 📌 Overview
A Power BI dashboard designed to track and visualize
OLA ride-hailing data — from booking performance to
revenue trends and customer satisfaction.

## 🎯 Key Features
- Completed vs. lost booking analysis
- Revenue trends by month & vehicle type
- Top pickup & drop location insights
- Customer & driver rating tracking
- Interactive vehicle type filters

## 🛠️ Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Power Query
